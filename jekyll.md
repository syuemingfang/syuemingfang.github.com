## Get Jekyll

 		$ gem install jekyll



## Create Jekyll Repo

 		$ jekyll new [Repo]



## Fork reveal.js

 		$ git clone https://github.com/hakimel/reveal.js.git



## Run Serve

 		$ jekyll serve
